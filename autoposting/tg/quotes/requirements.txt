# -----------------------------------------------------------------------------
# Планировщик задач
# -----------------------------------------------------------------------------
# Schedule - простая in-process библиотека для планирования задач
# Используется для ежедневной публикации цитат в случайное время
#
# Особенности:
#   - Не требует внешних зависимостей (брокеров сообщений)
#   - Работает в том же процессе
#   - Поддерживает сложные расписания
#
# Документация: https://github.com/dbader/schedule
#
# Пример использования:
#   schedule.every().day.at("09:00").do(send_quote)
#   schedule.every(10).minutes.do(job)
#
schedule==1.2.0
